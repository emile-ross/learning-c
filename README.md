### **For Linux (Ubuntu/Debian-based):**

#### **Install GCC:**
1. **Update your package list**:
   Debian:
   ```bash
   sudo apt update
   ```
   Arch Linux:
   ```bash
   sudo pacman -Syu
   ```

3. **Install GCC**:
   
   GCC is usually installed on all modern linux distros
   Debian:
   ```bash
   sudo apt install build-essential
   ```
   Arch Linux:
   ```bash
   sudo pacman -S base-devel
   ```
   This will install GCC along with other essential development tools.

5. **Verify GCC installation**:
   ```bash
   gcc --version
   ```
   This will display the installed version of GCC.
   If it says "unknown command" you will have to try installing it another way
